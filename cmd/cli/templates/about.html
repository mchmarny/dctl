About

dctl is an open source project created to provide a quick insight into the activity of a single repo or an entire GitHub organization across the two main dimensions:

  • Volume of developer events over time (PR, issue, and their comments)
  • Contributions by developer entity affiliation during time period (company or organization)
While GitHub does provide some charts with this information on a repo level, and some foundations created their own views into their projects, there isn't really anything free out there to provide answers to these questions.

Using data aggregated from your selection of GitHub API repositories (requires a free API access token), and augmented with developer affiliation content from CNCF and Apache Foundation, dctl provides organization, repository, or entity scoped drill-downs to filter on time period, contribution type, and developer name to identify individual events, and, optionally navigate to the original detail in GitHub for more context.

And, since all this data is cached locally, you can use a GUI (e.g. SQLiteStudio) or sqlite3 CLI to run the even more customized queries using SQL without needing to re-download the data. See the dctl readme file for more details on how to do that.

Hope you find this tool helpful. Let me know if you have any questions.
